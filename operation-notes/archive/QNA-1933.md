# QNA-1933: 급여정산 해지 시 완료한 정산의 명세서 공개/알림 동작 문의

> **상태**: 미착수 — 2026-03-18

## 증상
- **문제 정의**: 급여정산 구독 해지 후, 구독 중 완료한 정산의 명세서 공개일이 해지일 이후일 때 공개/알림이 정상 동작하는지 확인 필요
- **요청자**: 김한나 (CS)
- **영향 범위**: 급여정산 구독 해지 예정인 고객사
- **문제 시점**: 구독 만료일 이후 명세서 공개일이 도래하는 경우 (예: 만료일 24일, 공개일 25일)
- 문의 내용:
  1. 급여정산 구독 해지 후에도 구독 중 완료한 정산의 명세서 공개 및 알림이 동작하는가?[^1]
  2. 명세서 공개 기능을 사용하려면 구독을 연장해야 하는가?[^2]

## 현재까지 파악된 내용

### 현재 구현 동작 (김주원 확인)
- payroll에서 급여명세서 알림 발송 시 급여정산 구독 여부를 체크하는 로직 없음[^3]
- 실제 알림 발송은 pavement 서버에서 수행하는데, pavement에서도 구독 여부 체크 로직 없음[^3]
- **결론**: 구독 해지 후에도 완료한 정산의 명세서 공개일이 되면 웹/모바일/이메일 알림은 발송됨[^3]

### 미확인 사항
- 구독 해지 시 웹 화면/모바일에서 급여 탭 권한이 막히는데, 알림 클릭 후 동작은 미확인[^4]

### 제품 정책 방향 (이윤선 의견)
- 구독 해지 후 공개 기능 사용을 원하면 연장을 안내하는 것이 정책적으로 맞음[^5]
- 급여 오류 발견 시에도 수정할 방법이 없으므로 공개 이후까지 1달 연장 권장[^6]

### 고객 답변안 (이윤선 제안)
1. 해지 이후에도 완료한 정산의 급여명세서 공개 일정이 되면, 웹/모바일/이메일로 알림은 발송됨[^7]
2. 구독 해지 시 알림을 클릭해도 프로필을 통한 급여 조회는 차단되어 구성원이 실제 급여를 확인하는 데 문제 발생 가능[^7]
3. 급여 오류 발견 시에도 수정 불가하므로 급여 공개 이후까지 1달 사용 연장 권장[^7]
