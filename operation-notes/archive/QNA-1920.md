# QNA-1920: 연차 사용 내역 사라짐 — 연차 정책 부여 시작일 이전 데이터 미표시 (스펙)

> **상태**: 미착수 — 2026-03-18

## 증상
- **문제 정의**: 2024년 연차 사용 내역이 연차상세 페이지에서 보이지 않음 (엑셀 다운로드에는 존재)
- **회사**: 주식회사그레이스유 (Customer ID: 211315)[^1]
- **요청자**: 김혜인 (CS)
- **영향 범위**: 해당 고객사 전체 (연차 정책 설정에 따른 전사 영향)
- **문제 시점**: 2024년 연차 사용 내역
- 문의 내용:
  1. 휴가사용내역 엑셀에는 사용내역이 있는데, 연차상세 페이지에서는 2024년도 연차가 보이지 않음[^1]

## 현재까지 파악된 내용

### 원인 (안희종 확인)[^2]
- 구성원에게 매핑된 연차 정책의 **부여 시작일**이 2025년 1월로 설정되어 있음[^2]
- 부여 시작일 이전의 사용 내역은 제품 내에서 **없는 것처럼 취급**하는 것이 스펙[^2]
- Metabase 히스토리(#5078) 확인 결과, 처음부터 2025년 1월 1일로 설정되어 있었음[^3]

### 결론
- **스펙에 의한 정상 동작**. 엑셀에는 원시 데이터가 남아 있지만, 연차상세 페이지는 부여 시작일 이후만 표시.
