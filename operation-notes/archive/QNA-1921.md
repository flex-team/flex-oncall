# QNA-1921: 근로소득 지급명세서 누락분 팩스 제출 방법 문의

> **상태**: 미착수 — 2026-03-18

## 증상
- **문제 정의**: 근로소득 지급명세서 신고 시 중도 퇴사자 2명 누락 → 세무서 팩스 접수용 파일 형식 문의
- **요청자**: 김한나 (CS)
- **영향 범위**: 특정 고객사 (25년 중도 퇴사자 2명)
- **문제 시점**: 2026-03-16 (신고 기한 관련)
- 문의 내용:
  1. flex에서 근로소득 지급명세서는 신고 파일로만 생성되는데, 팩스로 보낼 수 있는 형태가 있는가?[^1]
  2. 누락 2명분을 추가 제출하는 경우 어떤 파일로 전달 가능한가?[^1]

## 현재까지 파악된 내용

### 답변 (최선아)[^2]
- 국세청에 근로소득지급명세서 양식이 한글파일로 업로드되어 있음[^2]
- 해당 양식을 수기 작성하여, 누락된 2명의 **개인별 원천징수영수증 수정본**과 함께 제출[^2]
- 세무서가 팩스 접수 후 내부 시스템에 수기 입력하는 프로세스[^3]

### 제출 절차 정리
1. 국세청에서 근로소득지급명세서 한글 양식 다운로드
2. 누락된 2명분 양식 수기 작성
3. 개인별 원천징수영수증 수정본 준비
4. 세무서에 팩스 접수 → 가산세 면제 가능[^1]
