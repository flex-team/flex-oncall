# CI-4280: 팬세이션 구성원 로그인 오류로 접속 불가

> **상태**: 원인 파악 완료 (온보딩 미완료), 고객 응답 대기 — 2026-04-02

## 증상

- **문제 정의**: 팬세이션주식회사 구성원(jinkim@fansation.net)이 PC 웹에서 로그인 시 오류로 접속 불가
- **회사**: 팬세이션주식회사 (Customer ID: 228464)
- **요청자**: yeogyung@flex.team
- **대상자**: jinkim@fansation.net
- **영향 범위**: 1명 (추가 확인 필요)
- **문제 시점**: 미기재
- 문의 내용:
  1. 등록 구성원이 로그인 시 오류로 접속되지 않음

## 원인 분석

**버그 아님. 해당 구성원이 초대 수락(온보딩) 미완료 상태.**

- Linear 라벨: 연동/보안/인증/성능/홈피드/공지[^1]
- 도메인: `:account` (계정/구성원) — 로그인 오류이므로 인증 흐름 문제 가능성[^2]
- 원인 파악 (2026-04-02, 김영준):
  - 구성원 등록은 되어 있으나 비밀번호 설정 단계(온보딩)를 거치지 않아 인증 계정이 미생성된 상태[^3]
  - 초대 메일 수락 → 비밀번호 설정 완료 시 로그인 가능
  - 고객에게 온보딩 완료 안내 요청 후 응답 대기 중
