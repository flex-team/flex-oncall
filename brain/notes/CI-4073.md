# CI-4073: 중도정산 금액 수정 후 2025년 연말정산 금액이 0원으로 반영됨

> **상태**: Todo — 2026-03-09 (KST)
> **verdict**: investigating — 중도정산 후 금액 수정 시 이전 연도 정산금액 소실

## 증상

- **문제 정의**: 2026년 2월 퇴사자 중도정산 중 문제 발생. 중도정산 완료 후 4단계 정산검토 화면에서 금액 수정 → 다시 중도정산 단계로 진행 시 2025년 사회보험 연말정산 금액이 사라짐. 2026년 소득 수정 시에도 2025년 정산금액이 0원으로 반영됨. 구체적으로 납부 **할** 보험료와 납부 **한** 보험료가 동일해지면서 정산금이 0원이 되는 현상.
- **회사**: 바이엘코리아 (Customer ID: 199890)
- **요청자**: PP
- **라벨**: 퇴직소득
- **담당자**: 이한결
- **Linear**: [CI-4073](https://linear.app/flexteam/issue/CI-4073/중도정산-금액-수정-후-2025년-연말정산-금액이-0원으로-반영됨)

## 원인 분석

- 이한결이 dev 환경에서 재현 시도했으나 재현 실패
- 박건남이 재현 영상을 DM으로 전달
- 4단계 정산검토에서 금액을 바꾸고 나면, 기존에 차이가 있었던(정산금이 발생하던) 납부할/납부한 보험료의 금액이 동일해지면서 정산금이 0원이 됨
- 박건남도 이후 재현 시도 시 재현 불가
- 코드 레벨 원인 분석을 위해 정산 결과 탭의 변경 전후 금액 정보를 요청했으나, 캡처 없이 기억에 의존한 정보만 확보됨
- 근본 원인 미파악 상태

## 해결

미해결. 재현이 안정적으로 되지 않아 코드 레벨 분석이 진행되지 못함.
