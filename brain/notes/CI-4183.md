# CI-4183: 리뷰 350건 이상 일괄 다운로드 지원 요청

> **상태**: 진행 중 — 2026-03-23

## 증상
- **문제 정의**: 마감 리뷰 약 350건을 일괄 다운로드하고 싶으나, 현재 건바이건 다운로드만 가능
- **회사**: 올리브영 (Customer ID: 156940)[^8]
- **요청자**: CS팀 나유라 → 고객 irene.seong@cj.net[^1]
- **대상자**: 올리브영 HR 담당자 (특정 구성원 아님)[^1]
- **영향 범위**: 올리브영 전사 마감 리뷰 전체[^1]
- **문제 시점**: flex 해지 예정으로 백업 필요[^1]
- 문의 내용:
  1. 마감 리뷰 약 350건을 한번에 다운로드할 수 있는지[^1]
  2. 제품 내 경로(리뷰 > 리뷰관리 > 리뷰 클릭 > 점세개 > 결과 다운로드)는 건바이건만 지원[^2]

## 현재까지 파악된 내용

### 다운로드 API 구조

사용자 화면의 다운로드 경로에 대응하는 API와 operation API를 확인했다.

| API | 경로 | 반환 타입 | 파일 다운로드 |
|-----|------|----------|-------------|
| 사용자용 (v2) | `GET /action/v2/review/sets/{idHash}/excel` | `ResponseEntity<Resource>` | **가능** |
| 사용자용 (v3) | `GET /action/v3/review/completed-sets/{idHash}/download-user-grade-by-excel` | `StreamingResponseBody` | **가능** |
| **Operation** | `GET /api/v2/operation/review/customers/{customerId}/sets/{reviewSetId}/excel` | **void** | **불가** ⚠️ |

> **핵심 발견**: Operation API가 `reviewSetExcelService.getReviewSetAsExcel()`을 호출하지만 반환값을 버린다[^3]. 파일 다운로드 용도로 사용할 수 없다.

### Operation API 파라미터

| 파라미터 | 타입 | 필수 | 비고 |
|---------|------|------|------|
| `customerId` | Long | O | 경로 변수 |
| `reviewSetId` | Long | O | 경로 변수 (action API는 HashedId — 해시 인코딩된 외부용 ID — 사용) |
| `userId` | Long | O | 쿼리 파라미터 |

### 마감 리뷰셋 확인

DB 조회로 올리브영의 마감 리뷰셋 존재를 확인했다[^4].

<details>
<summary>DB 조회 결과 (샘플 1건)</summary>

```sql
SELECT rs.id AS reviewSetId, rs.customerId, rs.title, rs.progressStatus
FROM review_set rs
WHERE rs.customerId = 156940
  AND rs.progressStatus IN ('ANSWER_NOT_OPEN', 'ANSWER_OPEN')
  AND rs.deleted = false
LIMIT 1;
```

| reviewSetId | customerId | title | progressStatus |
|-------------|-----------|-------|----------------|
| 15180 | 156940 | 올리브영 테스트 (셀프 리뷰 파일 업로드형식) | ANSWER_NOT_OPEN |

- `progressStatus`는 `ANSWER_NOT_OPEN`(답안공개 전) 또는 `ANSWER_OPEN`(답안공개)이 마감 상태[^5]

</details>

### 벌크 다운로드 API 부재

350건 리뷰셋을 한번에 다운로드하는 벌크 API는 존재하지 않는다[^6]. 단건 API를 반복 호출해야 한다.

## 발견한 스펙/제약
- Operation API(`/api/v2/operation/.../excel`)가 void를 반환하여 파일 다운로드에 사용할 수 없음[^3]
- 마감 상태: `ANSWER_NOT_OPEN`, `ANSWER_OPEN` (ReviewProgressStatus.FINISHED_STATUSES)[^5]
- 사용자용 API는 HashedId(내부 Long ID를 해시 인코딩한 외부 노출용 식별자), operation API는 Long ID를 사용 — ID 체계가 다름[^7]

## 용어 발견

| 사용자 표현 | 시스템 용어 | 비고 |
|------------|------------|------|
| 리뷰 다운로드 | ReviewSetExcelService.getReviewSetAsExcel() | 리뷰셋 단위 엑셀 다운로드 |
| 마감 리뷰 | progressStatus IN (ANSWER_NOT_OPEN, ANSWER_OPEN) | ReviewProgressStatus.FINISHED_STATUSES |
