## 원인 분석

- 진재언 님이 이슈 대상자의 연차 촉진 상태 확인:
  - `sy.hong7971@chabot.co.kr` → 모든 촉진문서 작성 완료 상태
  - `jh.yoon1324@chabot.co.kr` → 촉진문서 없음
  - 두 구성원 모두 메일 발송 조건에 해당하지 않음
- 메일 내 링크가 가리키는 유저 이메일은 **`people@chabot.co.kr`** — 이슈 대상자 목록에 없는 이메일
- `people@chabot.co.kr`은 **그룹 이메일(공용 메일)**로 확인됨
- 이 그룹 이메일이 flex 제품에 구성원으로 등록되어 있고, 연차 촉진 기록이 생성되어 있었음 (미작성 상태)
- 연차 촉진 메일은 "촉진 문서가 생성된 상태이고 아직 사용계획을 제출하지 않은 경우 구성원 자신에게 발송"되는 이메일
- 그룹 이메일로 수신한 메일의 링크를 개인 계정으로 클릭하면 → 서로 다른 계정이라 오류 페이지 노출
