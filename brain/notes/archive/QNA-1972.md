# QNA-1972: 구성원별 마지막 접속 기록 데이터 추출

> **상태**: 완료 — 2026-04-02

## 증상
- **문제 정의**: 활성 구성원의 마지막 접속 기록을 데이터로 추출할 수 있는지 문의
- **회사**: 미공개 (내부 감사 진행 중인 고객사)
- **요청자**: 권오은 (CS)
- **대상자**: 해당 회사의 전체 활성 구성원
- **영향 범위**: 전사 (활성 구성원 전원)
- **문제 시점**: 2026-04-01
- 문의 내용:
  1. 등록된 활성 구성원의 마지막 접속 기록을 추출할 수 있는가?
  2. 현재 감사로그에는 조회/생성/수정만 제공되는데, 접속 기록 데이터 제공이 가능한가?

## 원인 분석

### 조사 과정
> 💡 **배경**: 고객사 내부 감사 중 정보보안부서에서 "마지막 접속 기록" 데이터를 요청받은 상황[^1]
> → 고객사는 이미 flex 감사로그 데이터를 정보보안부서에 제출했으나, 추가로 접속 기록 제공 가능 여부 확인 요청[^2]

> 💡 **"마지막 접속"의 정의 모호성**: 논의 과정에서 "마지막 접속"이 무엇인지 정의가 불명확했음[^3]
> → 마지막 로그인 시점? 마지막 API 호출 시점? 마지막 세션 종료 시점?
> → 고객사의 의도는 "로그인 후 flex를 사용하다가 마지막으로 떠난 시점"으로 추정[^4]

### 시스템 현황 확인

| 데이터 소스 | 존재 여부 | 내용 | 한계 |
|------------|----------|------|------|
| `user_login_history` 테이블 | 존재 | 로그인 시도 기록 (성공/실패) | **로그인만 기록**, 이후 활동 추적 안 함[^5] |
| `user.z_last_login_at` 컬럼 | **DROP됨** | 2024-10-16에 제거[^6] | 사용 불가 |
| 감사로그 | 존재 | 조회/생성/수정/인증 이벤트 | 인증은 패스워드 인증만 남김[^7] |
| 유저별 마지막 access log 시간 | **없음** | — | 별도 테이블/컬럼 없음[^7] |
| raccoon auth Operation API | **없음** | 해당 오퍼레이션 미존재[^8] |

### 원인
> 💡 위 시스템 현황 확인 결과, 모든 후보 데이터 소스를 검토했으나 유저별 "마지막 접속 시간"을 저장하는 곳이 없음

- `login_history` 는 로그인 시도만 기록하고, 이후 활동은 추적하지 않음[^5]
- `user.z_last_login_at` 컬럼은 2024-10-16에 DROP됨[^6]
- access log의 유저별 마지막 시간을 업데이트하는 별도 테이블/로직이 없음[^7]

## 해결
- **직접 제공 불가**: 마지막 접속 기록 데이터는 시스템에 저장하지 않으므로 추출 불가[^7]
- **대안 안내**: 감사로그에서 최근 로그를 엑셀로 다운로드 → 사용자별 피벗을 돌려 "마지막 활동 시점"을 간접적으로 확인하는 방법을 안내[^9]

## 발견한 스펙/제약
- 감사로그의 "인증" 카테고리는 패스워드 인증만 기록하며, 토큰 갱신이나 SSO 인증은 포함하지 않음[^7]
- `login_history` 테이블이 감사로그보다 로그인 범위가 넓음 (패스워드 외 인증 방식 포함)[^5]

## 용어 발견
| 사용자 표현 | 시스템 용어 | 비고 |
|------------|------------|------|
| 마지막 접속 기록 | — (해당 데이터 없음) | 감사로그의 마지막 이벤트 시각으로 대체 가능 |
| 감사로그 | audit_log | 관리자 메뉴에서 다운로드 가능 |

## 다음에 같은 문의가 오면

1. **먼저 확인**: 고객이 원하는 "접속 기록"의 정확한 정의 파악 (마지막 로그인? 마지막 활동? 세션 종료?)
2. **원인 판별**:
   - "마지막 로그인" → 감사로그에서 인증 이벤트로 확인 가능
   - "마지막 활동(접속)" → 별도 저장 데이터 없음, 감사로그 피벗으로 간접 확인만 가능
3. **조치**: 감사로그 엑셀 다운로드 → 사용자 ID별 마지막 이벤트 시각을 피벗으로 추출하도록 안내. 향후에도 계속 필요한 데이터라면 배경과 함께 VOC로 등록 요청[^9]
