# CI-4099: 휴일대체 취소가 되지 않는 점 확인 요청

> **상태**: Done — 2026-03-12 (KST)
> **verdict**: bug — 휴일근무 삭제 후 휴일대체 취소 버튼 미노출

## 증상

- **문제 정의**: 2/1(일) 주휴일 근무의 휴일대체를 2/20(금)으로 지정 후, 2/1 근무를 삭제했으나 2/20은 휴일대체일로 유지됨. 2/1에 휴일대체 취소 버튼이 보이지 않아 2/20을 근무일로 되돌릴 수 없음. 보상휴가 부여 + 근태마감으로 잠긴 상태.

- **담당자**: 진재언

## 원인 분석
