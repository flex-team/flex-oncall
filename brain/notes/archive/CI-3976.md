# CI-3976: 퇴사자의 휴가 사용일자 데이터 추출 요청

## 증상
- **회사**: 주식회사 알트나인 (Customer ID[^10]: 52082)[^5][^6]
- **문의자**: hckim@alt9.co.kr[^5]
- **대상 구성원**: 퇴직자 전원[^11]
- 문의 내용:
  1. 플렉스 사용 이후 발생한 퇴사자의 휴가 사용 일자 데이터 추출 요청[^1]
  2. 사용 일수가 아닌, **언제 어떤 휴가를 사용했는지** (날짜 + 휴가 종류) 확인 필요[^1]
  3. 이용 기록이 4년치로, 근무 기록 화면에서 확인하기에 제한이 있음[^1]
  4. 인사발령 등으로 사라진 조직에 속한 퇴직자가 웹 UI에서 조회 대상으로 잡히지 않음[^12]

---

## 원인 분석

### 조사 과정

> 💡 **판단 근거**: 제품(웹 UI)에서 퇴직자 포함 다운로드가 가능한데 왜 안 되는지 확인[^13]
> → 고객사에서 인사발령으로 사라진 조직에 속한 구성원이 있어 조회 대상으로 잡히지 않음[^12]
> → 예시 유저: junmail@alt9.co.kr — 주조직[^14]이 없어서 대상자 목록에 미표시[^15]
> → 웹 UI는 조직 기반으로 구성원을 필터링하므로, 주조직 없는 퇴직자는 조회 불가[^13]
> → Operation API[^16]는 `departmentIds: null`로 조직 필터 없이 전체 조회 가능[^3]

### 원인
- 웹 UI의 휴가 사용 현황 다운로드는 **조직 기반 필터링**을 사용한다[^13]. 인사발령으로 주조직이 삭제된 퇴직자는 어떤 조직에서도 조회되지 않아 다운로드 대상에서 누락된다[^12].

---

## 해결
- Operation API로 퇴직자 포함 + 조직 필터 없이 전체 데이터를 엑셀로 추출하여 DM으로 전달[^17]

실제 호출한 API 정보[^18]:
```
POST /action/operation/v2/time-off/customers/52082/time-offs/excel/used
```
```json
{
  "queryUserId": 223046,
  "departmentIds": null,
  "dateFrom": "2023-02-01",
  "dateTo": "2026-03-01",
  "includeResignatedUsers": true
}
```

> 💡 `departmentIds: null`이 핵심이다. 조직 필터를 걸지 않으면 주조직 없는 퇴직자도 포함된다[^18].

---

## 다음에 같은 문의가 오면

1. **먼저 확인**: 웹 UI(휴가 관리 > 사용 내역 > 엑셀 다운로드)에서 "퇴직자 포함" 옵션으로 다운로드가 되는지 확인[^13]. 특정 구성원만 누락된다면 해당 구성원의 주조직 존재 여부를 확인한다.
2. **원인 판별**:
   - 주조직이 있는 퇴직자 → 웹 UI에서 정상 다운로드 가능[^13]. 고객에게 셀프 다운로드 안내
   - 주조직이 없는 퇴직자 → 웹 UI로 불가[^12]. Operation API 사용 필요[^3]
3. **조치**: Operation API 호출
   ```
   POST /action/operation/v2/time-off/customers/{customerId}/time-offs/excel/used
   ```
   ```json
   {
     "queryUserId": "<조회자 userId>",
     "departmentIds": null,
     "dateFrom": "<시작일>",
     "dateTo": "<종료일>",
     "includeResignatedUsers": true
   }
   ```
   - `queryUserId`: Metabase에서 고객사 관리자의 userId 조회[^9]
   - `departmentIds: null`: 조직 필터 미적용 (주조직 없는 퇴직자 포함)
   - 반환: 웹 다운로드와 동일한 형태의 엑셀 파일[^19]

> ⚠️ 우회 방법으로 "퇴직자에게 임시로 조직을 넣고 다운로드 후 다시 빼기"도 가능하나[^20], Operation API가 더 간편하다.

### Operation API 전체 목록 (휴가 사용 관련)

| 엔드포인트 | 기능 | 비고 |
|-----------|------|------|
| `POST /action/operation/v2/time-off/customers/{customerId}/time-offs/excel/used` | 휴가 사용 현황 엑셀 다운로드 | **퇴사자 포함 가능** (`includeResignatedUsers`)[^3] |
| `GET /api/operation/v2/time-tracking/time-off/customers/{customerId}/users/{userId}/annual-time-off-status/by-search-year/{searchYear}` | 구성원별 연차 부여/소멸/조정/사용 현황 조회 | 연차 요약 정보[^7] |
| `POST /api/operation/v2/time-tracking/time-off/customers/{customerId}/annual-time-off-history` | 연차 이력 조회 | email 또는 userId로 조회 가능[^7] |
| `POST /api/operation/v2/time-tracking/time-off/customers/{customerId}/annual-time-off-history.csv` | 연차 이력 CSV 다운로드 | 위와 동일, CSV 형태[^7] |
