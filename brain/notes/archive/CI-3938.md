# CI-3938: 중간정산 이력 시 퇴직사유가 중간정산으로 잘못 신고됨

> **상태**: Done — 2026-02-22 (KST)
> **verdict**: bug — 퇴직소득 지급명세서 신고 시 퇴직사유가 중간정산으로 잘못 표기

## 증상

- **문제 정의**: 퇴직소득 지급명세서 신고 시 중간정산 이력이 있는 경우 퇴직사유가 '중간정산'으로 신고됨. 원천징수영수증에서도 동일하게 '중간정산'으로 표기. 고객사 엘지유플러스볼트업 (ID: 191148).
- **라벨**: 퇴직소득, 급여/원천세/전표
- **담당자**: 이한결
- **Linear**: [CI-3938](https://linear.app/flexteam/issue/CI-3938/중간정산-이력-시-퇴직사유가-중간정산으로-신고됨)

## 원인 분석

중간정산 이력이 있는 퇴직자의 퇴직소득 지급명세서 생성 시, 퇴직사유 코드를 결정하는 로직이 중간정산 이력 유무만으로 사유를 판단하여 실제 퇴직 사유가 아닌 '중간정산'으로 설정한 것으로 추정.

## 해결

해결 완료 (상세 미기록)
