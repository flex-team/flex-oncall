# CI-4009: 허용된 휴가 사용 시간 범위를 벗어났다 오류 후 휴가가 등록됨

> **상태**: Done — 2026-03-04 (KST)
> **verdict**: not-a-bug — 오류 로그 미발견, 본인이 등록 후 취소한 이력 확인

## 증상

- **문제 정의**: 구성원이 2/24~25 새벽근무(25일 오전 7시까지) 후, 25일 정규 근무에 연차를 사용하려 했으나 "허용된 휴가 사용 시간 범위를 벗어났다" 오류 발생하여 휴가 등록 불가(구성원/관리자 모두). 그런데 3/3(월) 14시경 관리자가 구성원 프로필 > 보유한 연차 > 휴가 등록 경로로 시도하니 정상 등록됨.

## 원인 분석

- 진재언이 확인한 결과, **휴가사용시간 범위 관련 오류 로그가 발견되지 않음**
- 스크린샷을 확인한 결과, **본인이 등록을 했다가 취소한 이력이 있었음**
- 현재 동작: 근무기록이 깔려 있더라도 휴가 메뉴를 통해 휴가를 등록하면 근무를 덮은 상태로 휴가가 등록되는 것이 정상 동작
- 금요일에 안 되고 월요일에 된 이유는 명확하지 않으나, 사용자가 다른 경로로 시도했을 가능성

## 해결

- 현재 정상 동작 확인 후 Done 처리
- CX 담당자(이주화)에게 현재 동작 방식 전달
