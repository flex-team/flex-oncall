# CI-3880: 휴게 기록 요청 알림 텍스트 오류

> **상태**: Done — 2026-02-11 (KST)
> **verdict**: not-a-bug — 같은 사람이 보내고 받은 상황으로, 실제 버그 아님

## 증상

- **문제 정의**: 테스트 계정으로 근태 관리 > 휴게 시간 부족 구성원에게 휴게 기록 요청 시, 요청한 관리자 계정에 알림이 "(강호동 닉네임님이 -> 님에게)" 형태로 수신자 이름이 누락되어 표시됨.
- **라벨**: Tracking Squad, bug, 근태/근무/휴가
- **담당자**: 안희종
- **Linear**: [CI-3880](https://linear.app/flexteam/issue/CI-3880/테스트-계정으로-근태-관리-휴게-시간-부족-구성원에게-휴게-기록-요청을-했는데요-n요청한-관리자-계정에-알림이-이렇게-왔는데)

## 원인 분석

- 안희종이 확인한 결과: **같은 사람이 보내고 받은 상황**
- 보낸 쪽에는 푸시가 없고 받는 쪽 푸시만 있는 정상 동작
- 제보자(전수현)가 현재 접속한 계정이 "강호동 닉네임" 계정이었음 — 즉 자기 자신에게 요청을 보낸 것
- "호호호"가 회사명이지 계정명이 아니었음을 제보자가 혼동

## 해결

- 버그가 아닌 것으로 확인되어 즉시 Done 처리
- 같은 사람이 요청을 보내고 받았기 때문에, 받는 쪽 알림에서 "님에게"로 표시된 것은 자기 자신에 대한 알림이었음
