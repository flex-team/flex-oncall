# CI-4156: 근무유형 변경 예약 취소 불가

- 동일 이슈가 이미 다른 건으로 인입되어 있었음 (이지선 확인, [관련 스레드](https://flex-cv82520.slack.com/archives/CRU35U9FC/p1773813100570109))
