# CI-4052: 모바일 앱 출퇴근 시 간헐적 네트워크 오류 발생

> **상태**: Done — 2026-03-09 (KST)
> **verdict**: not-a-bug — 허용되지 않은 IP에서 출근 시도로 인한 403 오류, 사용자가 "네트워크 오류"로 오인

## 증상

- **문제 정의**: 안드로이드 앱(v2.120.1)에서 근무 시작 버튼 클릭 시 "네트워크 오류" 메시지가 간헐적으로 발생. 팀원 다수가 동일 증상 경험.

- **이슈 대상자**: limdk@blankd.net
- **라벨**: 모바일
- **담당자**: 미배정
- **Linear**: [CI-4052](https://linear.app/flexteam/issue/CI-4052/최근들어-모바일-앱에서-출퇴근-찍을-때-네트워크-오류가-간헐적으로-발생해요-팀원분들도-다-그렇다고-해서요-확인-부탁드립니다)

## 원인 분석

- 박민수가 해당 사용자의 오류 로그를 확인한 결과, 실제 발생한 오류는 **IP 접속 제한 (403)**:
  ```json
  {
    "type": "about:blank",
    "title": "Forbidden",
    "status": 403,
    "detail": "허용된 접속 환경이 아니에요. 허용된 IP에서 근무를 등록해주세요",
    "instance": "/api/v2/time-tracking/work-clock/users/YG0dVgYMEw/start",
    "code": "ACCESS_403_100"
  }
  ```
- 이택규: 회사에서 **와이파이 연결된 상태에서만 출근 가능하게 설정**해둔 것으로 추정
- 사용자가 허용되지 않은 IP(모바일 데이터 등)에서 근무 시작을 시도하여 403 오류가 반복 발생
- 사용자는 이 오류를 "네트워크 오류"로 표현한 것

## 해결

- 실제 시스템 오류가 아닌 **IP 접속 제한 설정에 의한 정상 동작**으로 확인
- CX 담당자에게 원인 전달
