# CI-4049: API 부서(departments) 호출 시 null 리턴

> **상태**: 해결 완료 (Not a Bug) — 2026-04-02
> **verdict**: not-a-bug — 고객사 설정 누락 (조직 코드 미입력)

## 증상

- **문제 정의**: Open API로 부서 정보 호출 시 부서 필드가 null(빈칸)으로 내려옴. HTTP 200 응답이며 사번 정보는 정상 반환되나 부서 정보만 일괄 누락. 구성원 프로필에는 부서가 정상 입력되어 있음.
- **회사**: 에스지에이 그룹사 (Customer ID: 208837)
- **요청자**: inhak2@sgacorp.kr
- **라벨**: 연동/보안/인증/성능/홈피드/공지
- **담당자**: 미배정
- **Linear**: [CI-4049](https://linear.app/flexteam/issue/CI-4049)

## 원인 분석

- DB 조회 결과, 해당 고객사의 조직 코드가 null임을 확인:
  ```sql
  select code from department where id = 208837
  -- 1건 / null
  ```
- Open API에서 부서 정보를 조회하려면 사전에 **조직 코드**를 등록해야 함
- 관련 문서: https://developers.flex.team/reference/faq-limitation#항목-별-사전-코드-등록-필요
- 고객사가 flex 설정에서 조직 코드 값을 입력하지 않은 것이 원인

## 해결

- **이슈 아님**: 고객사 설정 누락. 조직 코드를 입력하면 정상 작동.
- CS 담당자(김현욱)가 고객에게 조직 코드 등록 필요 안내 완료
