# CI-4199: 주휴일 휴일대체 지정 가능 기한 변경 (에이엠에이)

> **상태**: 해결 완료 — [config PR #40](https://github.com/flex-team/flex-timetracking-config/pull/40) 생성 (2026-03-24)

## 증상

- **문제 정의**: 주휴일 휴일대체 지정 가능 기간을 기본값(±6일)에서 **±180일**로 변경 요청
- **회사**: 에이엠에이 (Customer ID: `220755`)
- **요청자**: yuri.kim@ama-official.com (고객사), CS팀 이주화[^1]
- **영향 범위**: 해당 회사 전체
- **문제 시점**: 2026-03-24 인입
- 문의 내용:
  1. 주휴일의 휴일대체 지정일을 공휴일의 휴일대체 지정일과 같이 180일 전후로 변경 요청[^2]

> ⚠️ Intercom 원문에서는 "두달까지 입력 가능하도록"이라고 언급[^3]하나, CS팀이 Linear에 "180일 전후"로 정리함. Linear 기준으로 진행.

---

## 원인 분석

**스펙 이슈** — CI-3897, CI-4186과 동일 패턴. 주휴일 gap은 회사별 config로 변경 가능하다.

### 가설 목록

| # | 가설 | 확인 방법 | 상태 |
|---|------|----------|------|
| 1 | config 변경으로 해결 가능 (CI-3897/CI-4186 동일 패턴) | 쿡북 체크리스트 2번 + 선례 확인 | ✅ 확정 |

> 💡 **판단 근거**: CI-4186[^4]에서 동일 요청(한국로날드맥도날드하우스, ±30일)을 `experimental.json`의 `customGapOfWeeklyHoliday` 설정으로 해결한 선례 있음 → 동일 방식 적용

### 코드 위치

| 역할 | 위치 |
|-----|------|
| 동적 설정 로드 | `flex-timetracking-backend` > `applications/application-config/.../TrackingExperimentalDynamicConfig.kt` |
| 상수 정의 | `flex-timetracking-backend` > `time-tracking/service/.../alternativeholiday/commons/CommonConstants.kt` — `MAX_GAP_OF_WEEKLY_HOLIDAY_IN_DATES = 6L` |

### 스펙 vs 버그

**스펙** — 주휴일 gap은 회사별 커스텀 설정이 설계 시점부터 지원됨. 기본값(±6일)은 근로기준법 제55조에 근거[^5].

---

## 해결

`flex-timetracking-config` > `experimental.json` > `customGapOfWeeklyHoliday`에 `"220755": 180` 추가.

- [config PR #40](https://github.com/flex-team/flex-timetracking-config/pull/40) — draft 생성 완료 (2026-03-24)
- 머지 시 Central Dogma 1분 내 자동 반영[^6]

| 항목 | 내용 |
|-----|------|
| 작업량 | config 1줄 추가, 코드 변경 없음 |
| 변경 효과 | ±6일(기본) → **±180일** (총 361일 범위) |
| 배포 | 불필요 (Central Dogma 동적 반영) |
| 근로자 영향 | 선택 범위 확대 — **유리** ✅ |
| 법적 리스크 | 없음 — 근로자에게 유리한 방향[^5] |

## 다음에 같은 문의가 오면

1. **쿡북 체크리스트 2번** 확인 — "휴일대체 기간 문의 → `TrackingExperimentalDynamicConfig` gap 설정"
2. **CI-3897 선례** 참조 — 동일 패턴으로 config PR 생성
3. `flex-timetracking-config` > `experimental.json` > `customGapOfWeeklyHoliday`에 `"{customer_id}": {일수}` 추가
4. 고객에게 "전후 대칭만 가능"한 시스템 제약 안내 (비대칭 요구 시)
