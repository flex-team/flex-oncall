# CI-4071: 카드 부분 취소 내역이 전체 취소로 표시됨

> **상태**: Done — 2026-03-11 (KST)
> **verdict**: bug — 하나카드 부분취소 시 취소금액 필드 미사용으로 전체취소 금액이 영수증에 반영됨

## 증상

- **문제 정의**: 카드 사용 내역 중 부분 취소된 건이 전체 취소로 내역에 잡힘. 카드사(하나카드)에서는 부분 취소로 정상 표시되나, flex 비용관리에서는 전체 취소 금액으로 영수증이 생성됨.
- **회사**: 주식회사 주빅 (Customer ID: 223917)
- **요청자**: clkang@juvicbio.com
- **라벨**: 비용관리
- **담당자**: 김성훈
- **Linear**: [CI-4071](https://linear.app/flexteam/issue/CI-4071)

## 원인 분석

- 하나카드 승인 데이터에서 부분취소 시, **승인금액 필드**에는 전체 금액이, **취소금액 필드(nullable)**에 실제 부분취소 금액이 내려옴 (우리카드와 동일 구조)
- 기존 코드는 승인금액 필드만 사용하여 취소 금액을 판단했기 때문에, 부분취소 시에도 전체 금액이 취소 금액으로 처리됨
- 매입 데이터에는 아직 취소 내역이 반영되지 않은 상태
- 기존에 부분취소 케이스가 드물어 전체 취소만 테스트해서 미발견된 버그
- 해외 결제에서도 동일한 문제 확인됨

## 해결

1. **코드 수정 (hotfix)**: 하나카드 부분취소 시 취소금액 필드를 사용하도록 코드 수정 배포
2. **2차 hotfix**: operation 수행 중 에러 발생하여 추가 hotfix 수행 (제품 데이터에는 영향 없음)
3. **데이터 보정**: 기존 영수증(미제출 상태)을 삭제하고, 부분취소 금액이 반영된 새 영수증으로 재생성
4. **근본 처리**: 앞으로 하나카드 부분취소가 정상 반영되도록 영구 수정 완료
5. 노션 문서에도 카드사별 부분취소 처리 방식 업데이트됨
