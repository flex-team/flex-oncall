# CI-3568: 지방소득세 신고서 생성 시 법정동코드 오류

> **상태**: Done — 2026-02-13 (KST)
> **verdict**: not-a-bug — 법정동코드는 정상 생성되었으며, 위택스 신고도 문제 없이 완료

## 증상

- **문제 정의**: 지방소득세 신고서 생성 시 11월과 12월에 서로 다른 법정동코드가 생성됨. 11월은 `진위면(진위면) 4122031031027`, 12월은 `진위면(갈곶리) 4122231031027`로, 고객은 12월 코드가 조회되지 않는 코드라고 신고. 고객은 당시 수동 변경하여 제출 완료.

## 원인 분석

- 사업장 주소: 경기도 평택시 진위면 갈곶리 729-1 물류동 1층 (도로명: 경기도 평택시 진위면 진위2산단로 45)
- 현재 코드와 데이터 형상으로 볼 때, 신고관할지 코드 `진위면(갈곶리)_4122231031027`이 정상적으로 들어가는 것이 기대값 (김주원 확인)
- 위택스 엑셀 신고서 파일에도 `진위면(갈곶리)_4122231031027` 신고관할지코드가 정상 포함
- 고객이 검색한 사이트(code.go.kr)와 위택스가 사용하는 법정동코드 체계가 다를 수 있음
- 1월 귀속 신고서도 정상 완료 확인 (2026-02-10 신고기한 이후 확인)

## 해결

- 실제 버그가 아닌 것으로 판단: 위택스 신고에 문제 없이 접수 완료
- 서브이슈 [CI-3574](https://linear.app/flexteam/issue/CI-3574) 생성하여 추적
- 1월 신고 후 결과 확인(REVISIT) → 정상 완료 확인 후 티켓 종료
