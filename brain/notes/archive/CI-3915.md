# CI-3915: 참조 설정에 없는 구성원에게 워크플로우 알림 발송됨

> **상태**: Done — 2026-02-19 (KST)
> **verdict**: not-a-bug — CEO 조직에 "하위조직 포함" 설정이 되어 있어 정상 동작

## 증상

- **문제 정의**: 워크플로우 문서(문서번호 2026-317) 작성 시, 참조/승인자에 포함되지 않은 구성원(gy.yu@thefounders.kr, 운영HQ-법무팀 팀원)에게 참조 알림이 발송됨. 참조자를 여러 차례 수정했으며, CEO 직책에 해당 구성원이 포함되어 있지 않다고 주장.
- **회사**: 더파운더즈 (Customer ID: 60371)
- **요청자**: bh.jun@thefounders.kr
- **이슈 대상자**: gy.yu@thefounders.kr
- **라벨**: 워크플로우
- **담당자**: 전우균
- **Linear**: [CI-3915](https://linear.app/flexteam/issue/CI-3915)

## 원인 분석

- 데이터 확인 결과, **CEO 조직이 "하위조직 포함"으로 설정**되어 있었음
- CEO 조직의 하위조직까지 모두 참조 대상에 포함되므로, 법무팀 소속 구성원도 참조 알림을 받은 것
- 메타베이스 조회: target uid `55a3b84a8b2a424cbd14e34e963fe811`, 피플실 하위조직 포함 설정 확인
- 참조자 유저 목록에 gy.yu@thefounders.kr 메일이 직접 없지만, 하위조직 포함 옵션으로 인해 자동 포함됨
- 문의자가 델리(관리자 콘솔) 접근 불가하여 직접 설정 확인이 어려웠음

## 해결

- **이슈 아님**: 설정대로 정상 동작. CEO 조직에 "하위조직 포함" 옵션이 활성화되어 있어 하위 모든 조직 구성원에게 참조 알림이 발송된 것.
- 고객에게 참조 설정의 "하위조직 포함" 옵션 안내 완료
