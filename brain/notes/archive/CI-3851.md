# CI-3851: 배우자 출산휴가 일수가 특정 구성원에게 10일로 표기되는 버그

> **상태**: Done — 2026-02-10 (KST)
> **verdict**: user-action — 과거에 10일짜리 배우자 출산휴가를 사용 후 취소하여 10일 버킷이 잔존, 회수 후 재신청으로 해결

## 증상

- **문제 정의**: 배우자 출산휴가가 20일로 세팅되어 있으나, 특정 구성원(신권철)에게는 10일로 표기됨.
- **보고자**: 이경훈 (내부 피드백)
- **라벨**: Tracking Squad, bug, 근태/근무/휴가
- **담당자**: 김영준(Enhance)
- **Linear**: [CI-3851](https://linear.app/flexteam/issue/CI-3851/배우자-출산휴가가-20일로-세팅되어-있습니다n특정-구성원은-배우자-출산휴가가-20일이-아니라-10일로-표기가-됩니다)

## 원인 분석

- 이지선 님 추정 → 김영준 님 확인: 신권철 님이 과거(2022/2/10 17:44)에 배우자 출산휴가를 10일로 사용 후 취소한 이력이 있어 **10일짜리 버킷이 생겨서 잔존**
- 초산이 아니어서 당시 10일 기준으로 신청했던 것으로 확인
- 관련 Slack 스레드: [CKC2E3GJC 채널](https://flex-cv82520.slack.com/archives/CKC2E3GJC/p1770625510699299?thread_ts=1770625342.972729&cid=CKC2E3GJC)

## 해결

- **조치**: 관리자(이경훈)가 기존 10일짜리 배우자 출산휴가를 **회수** → 20일짜리로 **재신청**
- 회수 방법: 내 휴가 화면에서 해당 휴가 선택 후 회수 처리 (신권철 님에게는 회수 권한이 없어 관리자인 이경훈 님이 수행)
- 회수 후 정상적으로 20일로 변경 확인 완료

## 비고
