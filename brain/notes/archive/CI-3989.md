# CI-3989: 퇴직소득 지급명세서 신고파일 음수 금액 시 파일 생성 불가

> **상태**: Done — 2026-03-05 (KST)
> **verdict**: enhancement — 퇴직소득세 음수 시 신고파일 다운로드 불가, 홈택스는 음수 허용

## 증상

- **문제 정의**: 퇴직소득 지급명세서 신고파일 다운로드 시 음수 금액이 있으면 파일 생성이 불가. 홈택스에서 해당 인원에 대해 수기로 퇴직소득세를 음수로 입력하여 신고하니 접수 완료됨. 신고해야 하는 인원이 다수이므로 시스템에서 음수 상태로 파일 생성이 필요.
- **요청자**: 강선하 (CS 파트너)
- **라벨**: Payroll Squad, enhancement, 급여/원천세/전표
- **담당자**: 김동규
- **Linear**: [CI-3989](https://linear.app/flexteam/issue/CI-3989)

## 원인 분석

- 퇴직소득 지급명세서 파일 생성 로직에서 음수 값을 허용하지 않는 검증 로직 존재
- 근로소득 간이지급 명세서는 0원 이하 불가이나, 퇴직소득 지급명세서는 홈택스에서 음수 신고가 허용됨 (고객 확인: 홈택스 접수증 첨부)
- 김동규가 이 차이를 확인 후 퇴직소득 지급명세서에 한해 음수 허용 방향으로 결정

## 해결

- **코드 수정**: 퇴직소득 지급명세서 파일 생성 시 음수 금액도 허용하도록 변경
