# CI-3989: 퇴직소득 지급명세서 신고파일 음수 금액 시 파일 생성 불가

> **상태**: Done — 2026-02-26 (KST)

- **문제 정의**: 퇴직소득 지급명세서 신고파일 다운로드 시 음수 금액이 있으면 파일 생성이 불가. 홈택스에서 수기로 음수 입력하여 신고하면 접수 완료됨.
