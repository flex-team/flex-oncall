# CI-4315: 금융기관 연동 전 카드 내역 수집 요청

## 증상
- **문제 정의**: 새롭게 연동한 신한카드의 과거 사용내역(2026년 1월 한달)을 수집해달라는 운영 요청
- **회사**: 림피드 (Customer ID: 66365)
- **요청자**: heesu0815@rhinobox.co.kr[^1]
- **대상자**: 특정되지 않음 (회사 전체 카드 내역으로 추정)
- **영향 범위**: 1개 회사
- **문제 시점**: 2026년 1월 1일 ~ 1월 31일 (수집 희망 기간)
- 문의 내용:
  1. 새롭게 연동한 신한카드의 과거 사용내역 수집 요청[^1]
  2. 카드사: 신한카드[^1]
  3. 연동 요청 기간: 2026년 1월 1일 전체 한달[^1]

## 원인 분석

### 조사 과정
> 💡 금융기관 연동 이전 기간의 카드 사용내역은 자동 수집되지 않으므로, 수동 동기화 운영이 필요한 건이다.
> → 별도 조사 불필요 — 정상적인 운영 요청

### 원인
- 카드사 연동 시점 이전의 과거 데이터는 자동 수집 대상이 아님[^5]
- 과거 내역이 필요한 경우 어드민 쉘의 수동 동기화 기능으로 처리[^2]

## 영향 범위
- 림피드 1개 회사, 신한카드 2026년 1월분 한정
- 금전적/법적 영향 없음

## 해결
- 김성훈이 어드민 쉘의 수동 동기화 기능으로 1월 데이터 반영 완료[^2]
- 수동 동기화 기능: 고객사 + 금융기관 + 동기화 대상 + 동기화 일자 범위를 선택하여 실행[^2]
- 수동 동기화 완료 알림 확인됨[^3]
- CS 담당자(유지원) 확인 완료[^4]

## 다음에 같은 문의가 오면

1. **먼저 확인**: 해당 고객사의 카드사 연동 상태 및 연동 시점 확인
2. **원인 판별**: 연동 이전 기간 데이터 요청인지 확인 — 연동 이전이면 수동 동기화 필요
3. **조치**: 어드민 쉘에서 수동 동기화 실행 (고객사 + 금융기관 + 동기화 대상 + 일자 범위 선택). 상세 절차는 [runbook](https://www.notion.so/flexnotion/32c0592a4a928167aebcfcca508cc0ba?source=copy_link#32ded628dff04934aa6d3497d261f743) 참고[^2]
